# 🛍️ ShopEase AI Support Bot

An intelligent customer support bot built with Google Gemini AI and Streamlit.

## Features
- 🧠 Answers from a custom knowledge base
- 😤 Detects frustrated users and escalates
- 🤔 Knows when to hand off to a human agent
- 📊 Live analytics dashboard

## Tech Stack
- Python
- Google Gemini API
- Streamlit
- python-dotenv

## How to Run
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Add your Gemini API key to `.env` file as `GEMINI_API_KEY=your-key`
4. Run: `streamlit run app.py`
